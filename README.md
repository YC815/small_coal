# [Minecraft-Datapack] 小煤炭資料包 by YC815
## <b> 👉 簡介：<b/>
本作品靈感來源來自《實用擴展》中的小煤炭。<br>
您可以製作出小煤炭，一個小煤炭可以燒至一個物品。<br>
您也可以使用8個小煤炭合成出一個一般的煤炭。<br><br>

## <b> 👉 使用說明：<b/>
- 小煤炭合成配方（無序合成）：
`|#|`<br><br>
  > #: 煤炭<br>
- 使用方法：
  > 直接將小煤炭當作燃料即可<br>
